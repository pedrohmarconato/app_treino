1. exercicios
Descrição: Armazena informações sobre os exercícios disponíveis.
Colunas:
id (bigint, PK): Identificador único do exercício.
nome (text): Nome do exercício.
grupo_muscular (text): Grupo muscular que o exercício trabalha.
equipamento (text): Equipamento necessário para o exercício.
tempo_descanso_padrao (integer, default 0): Tempo de descanso padrão entre as séries.
created_at (timestamp with time zone, default now()): Data e hora de criação do registro.
updated_at (timestamp with time zone, default now()): Data e hora da última atualização do registro.

2. pesos_usuario
Descrição: Armazena os pesos utilizados pelos usuários em seus treinos.
Colunas:
id (serial, PK): Identificador único do registro.
usuario_id (integer, FK): Referência ao usuário.
protocolo_treino_id (integer, FK): Referência ao protocolo de treino.
peso_base (numeric(5,2)): Peso base utilizado.
peso_minimo (numeric(5,2)): Peso mínimo permitido.
peso_maximo (numeric(5,2)): Peso máximo permitido.
repeticoes_alvo (integer): Número de repetições alvo.
data_calculo (timestamp without time zone, default CURRENT_TIMESTAMP): Data do cálculo do peso.
status (character varying(10), default 'ativo'): Status do registro.

3. protocolo_treinos
Descrição: Armazena os protocolos de treino que associam exercícios a um plano de treino.
Colunas:
id (bigserial, PK): Identificador único do protocolo de treino.
protocolo_id (bigint, FK): Referência ao protocolo de treinamento.
exercicio_id (bigint, FK): Referência ao exercício.
numero_treino (integer): Número do treino.
semana_referencia (integer): Semana de referência do treino.
dia_semana (integer): Dia da semana do treino.
percentual_1rm_base (numeric(5,2), default 0): Percentual da 1RM base.
percentual_1rm_min (numeric(5,2), default 0): Percentual mínimo da 1RM.
percentual_1rm_max (numeric(5,2), default 0): Percentual máximo da 1RM.
series (integer, default 1): Número de séries.
repeticoes_alvo (integer, default 10): Número de repetições alvo.
tempo_descanso (integer, default 60): Tempo de descanso entre as séries.
ordem_exercicio (integer, default 1): Ordem do exercício no treino.
observacoes (text): Observações sobre o treino.
created_at (timestamp with time zone, default now()): Data e hora de criação do registro.
updated_at (timestamp with time zone, default now()): Data e hora da última atualização do registro.

4. protocolos_treinamento
Descrição: Armazena informações sobre os protocolos de treinamento.
Colunas:
id (bigint, PK): Identificador único do protocolo.
nome (text): Nome do protocolo.
descricao (text): Descrição do protocolo.
duracao_meses (integer, default 1): Duração do protocolo em meses.
total_semanas (integer, default 4): Total de semanas do protocolo.
dias_por_semana (integer, default 3): Dias de treino por semana.
total_treinos (integer, default 12): Total de treinos no protocolo.
status (text, default 'ativo'): Status do protocolo.
created_at (timestamp with time zone, default now()): Data e hora de criação do registro.
updated_at (timestamp with time zone, default now()): Data e hora da última atualização do registro.

5. usuario_1rm
Descrição: Armazena os dados de 1RM (uma repetição máxima) dos usuários.
Colunas:
id (bigserial, PK): Identificador único do registro.
usuario_id (bigint, FK): Referência ao usuário.
exercicio_id (bigint, FK): Referência ao exercício.
peso_teste (numeric(6,2)): Peso testado pelo usuário.
repeticoes_teste (integer): Número de repetições testadas.
rm_calculado (numeric(6,2)): 1RM calculado.
data_teste (date): Data do teste.
status (text, default 'ativo'): Status do registro.
created_at (timestamp with time zone, default now()): Data e hora de criação do registro.
updated_at (timestamp with time zone, default now()): Data e hora da última atualização do registro.

6. usuarios
Descrição: Armazena informações sobre os usuários do sistema.
Colunas:
id (bigint, PK): Identificador único do usuário.
nome (text): Nome do usuário.
email (text, unique): Email do usuário.
data_nascimento (date): Data de nascimento do usuário.
status (text, default 'ativo'): Status do usuário.
created_at (timestamp with time zone, default now()): Data e hora de criação do registro.
updated_at (timestamp with time zone, default now()): Data e hora da última atualização do registro.

Views
1. v_comparativo_usuarios
Descrição: Compara os 1RMs de diferentes usuários para um exercício específico.
Colunas:
exercicio: Nome do exercício.
grupo_muscular: Grupo muscular do exercício.
pedro_1rm: 1RM do usuário Pedro.
japa_1rm: 1RM do usuário Japa.
diferenca_kg: Diferença em kg entre os 1RMs de Japa e Pedro.
mais_forte: Usuário mais forte entre Pedro e Japa.

nome_exercicio: Nome do exercício.
